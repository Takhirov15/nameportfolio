что я использую при создания сайта?

Здесь я кратко расскажу что я использую!

Фундаментом канечно является HTML , и СSS что такое HTML? HTML это - стандартизированный язык разметки документов. На HTML создаётся контент для вашего сайта. Например: написать текст, добавить фото или видео и т.д.

Для чего нужен CSS для сайта?

CSS - язык для формирования внешнего вида сайта. Он отвечает за цвета, шрифты, размеры текстов и всего что касается стиля. Если представить строительство дома - то кирпичи, крыша, каркас это HTML. А вот обои, цвет стен, красивые двери, окна и всё то что придаёт нашему дому визуальную красоту - это уже CSS.

Для чего нужен PHP для сайта?

PHP - это язык программирования работающей со сервером. Он нужен для входа, регистрации аккаунта, и многое другое. По желанию могу использовать ещё JavaScript, и Woordprees если хотите !
